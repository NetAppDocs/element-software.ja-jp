= 証明書の詳細
:icons: font
:imagesdir: ../media/


[role="lead"]
certificateDetails オブジェクトには、セキュリティ証明書に関するデコードされた情報が含まれます。

このオブジェクトのメンバーは次のとおりです。

|===
| 名前 | 説明 | を入力します 


 a| 
発行元
 a| 
発行元の名前。
 a| 
文字列



 a| 
モジュラス
 a| 
公開鍵のモジュラス。
 a| 
文字列



 a| 
notAfter
 a| 
証明書の有効期限。
 a| 
ISO 8601 形式の文字列



 a| 
メモ前段
 a| 
証明書の開始日。
 a| 
ISO 8601 形式の文字列



 a| 
Serial （シリアル）
 a| 
証明書のシリアル番号。
 a| 
文字列



 a| 
sha1 指紋
 a| 
DER でエンコードされたバージョンの証明書のダイジェスト。
 a| 
文字列



 a| 
件名
 a| 
件名。
 a| 
文字列

|===
