= DisableSnmp
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
「 D isableSnmp 」メソッドを使用すると、クラスタノードの SNMP を無効にできます。



== パラメータ

このメソッドには入力パラメータはありません。



== 戻り値

このメソッドには戻り値はありません。



== 要求例

このメソッドの要求例を次に示します。

[listing]
----
{
  "method": "DisableSnmp",
  "params": {},
  "id" : 1
}
----


== 応答例

このメソッドの応答例を次に示します。

[listing]
----
{
  "result" : {},
  "id" : 1
}
----
