= GetBackupTarget
:icons: font
:imagesdir: ../media/


[role="lead"]

== 戻り値

このメソッドの戻り値は次のとおりです。

|===
| 名前 | 説明 | を入力します

== 要求例

このメソッドの要求例を次に示します。

[listing]
----
{
  "id": 1,
  "method": "GetBackupTarget",
  "params": {
      "backupTargetID": 1
  }
}
----


== 応答例

このメソッドの応答例を次に示します。

[listing]
----
{
   "id": 1,
   "result": {
     "backupTarget": {
     "attributes" : {
        "size" : 100
     },
     "backupTargetID" : 1,
     "name" : "mytargetbackup"
    }
  }
}
----
