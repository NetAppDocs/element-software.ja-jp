= GetClusterConfig
:icons: font
:imagesdir: ../media/


[role="lead"]
GetClusterConfig API メソッドを使用すると、ノードがクラスタとの通信に使用するクラスタ構成の情報を取得できます。



== パラメータ

このメソッドには入力パラメータはありません。



== 戻り値

このメソッドの戻り値は次のとおりです。

|===
| 名前 | 説明 | を入力します 


 a| 
クラスタ
 a| 
ノードがクラスタとの通信に使用するクラスタ構成情報。
 a| 
xref:reference_element_api_cluster.adoc[クラスタ]

|===


== 要求例

このメソッドの要求例を次に示します。

[listing]
----
{
   "method": "GetClusterConfig",
   "params": {},
   "id" : 1
}
----


== 応答例

このメソッドの応答例を次に示します。

[listing]
----
{
  "id": 1,
  "result": {
    "cluster": {
      "cipi": "Bond10G",
      "cluster": "ClusterName",
      "ensemble": [
        "1:10.30.65.139",
        "2:10.30.65.140",
        "3:10.30.65.141"
      ],
      "fipsDriveConfiguration": true,
      "mipi": "Bond1G",
      "name": "xxx-en142",
      "nodeID": 4,
      "pendingNodeID": 0,
      "role": "Storage",
      "sipi": "Bond10G",
      "state": "Active",
      "version": "9.1.0"
    }
  }
}
----
