= GetClusterInfo を使用します
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
GetClusterInfo' メソッドを使用すると、クラスタの構成情報を取得できます。



== パラメータ

このメソッドには入力パラメータはありません。



== 戻り値

このメソッドの戻り値は次のとおりです。

|===
| 名前 | 説明 | を入力します 


 a| 
clusterInfo
 a| 
クラスタ情報
 a| 
xref:reference_element_api_clusterinfo.adoc[clusterInfo]

|===


== 要求例

このメソッドの要求例を次に示します。

[listing]
----
{
   "method": "GetClusterInfo",
   "params": {},
   "id" : 1
}
----
