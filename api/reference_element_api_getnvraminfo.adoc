= GetNvramInfo をクリックします
:icons: font
:imagesdir: ../media/


[role="lead"]
GetNvramInfo' メソッドを使用すると 'NVRAM カードに関する各ノードから情報を取得できます



== パラメータ

このメソッドの入力パラメータは次のとおりです。

|===
| 名前 | 説明 | を入力します | デフォルト値 | 必須 


 a| 
[-force]
 a| 
このメソッドをクラスタのすべてのノードに対して実行するには、 force パラメータを指定する必要があります。
 a| 
ブール値
 a| 
なし
 a| 
はい。

|===


== 戻り値

このメソッドの戻り値は次のとおりです。

|===
| 名前 | 説明 | を入力します 


 a| 
nvramInfo
 a| 
NVRAM カードで検出されたイベントとエラーの配列。
 a| 
JSON オブジェクト

|===


== 要求例

このメソッドの要求例を次に示します。

[listing]
----
{
  "method": "GetNvramInfo",
  "params": {
      "force": true
	  },
  "id" : 1
}
----
