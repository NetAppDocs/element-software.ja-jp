= ListActiveNodes
:icons: font
:imagesdir: ../media/


[role="lead"]
ListActiveNodes メソッドを使用すると、クラスタ内で現在アクティブなノードのリストを取得できます。



== パラメータ

このメソッドには入力パラメータはありません。



== 戻り値

このメソッドの戻り値は次のとおりです。

|===
| 名前 | 説明 | を入力します 


 a| 
ノード
 a| 
クラスタ内の Active 状態のノードのリスト。
 a| 
xref:reference_element_api_node.adoc[ノード] 配列

|===


== 要求例

このメソッドの要求例を次に示します。

[listing]
----
{
   "method": "ListActiveNodes",
   "params": {},
   "id" : 1
}
----
