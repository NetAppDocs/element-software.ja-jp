= ListServices の場合
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ListServices メソッドを使用すると、ノード、ドライブ、現在のソフトウェア、およびクラスタで実行中のその他のサービスに関するサービス情報をリストできます。



== パラメータ

このメソッドには入力パラメータはありません。



== 戻り値

このメソッドの戻り値は次のとおりです。

|===
| 名前 | 説明 | を入力します 


 a| 
サービス
 a| 
ドライブおよびノードで実行されているサービス。
 a| 
JSON オブジェクト

|===


== 要求例

このメソッドの要求例を次に示します。

[listing]
----
{
   "method": "ListServices",
   "params": {},
   "id" : 1
}
----
