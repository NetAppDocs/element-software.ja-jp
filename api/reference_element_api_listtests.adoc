= ListTests のように指定
:icons: font
:imagesdir: ../media/


[role="lead"]
「 ListTests 」メソッドを使用すると、ノードで実行可能なテストを一覧表示できます。



== パラメータ

このメソッドには入力パラメータはありません。



== 戻り値

このメソッドの戻り値は次のとおりです。

|===
| 名前 | 説明 | を入力します 


 a| 
テスト
 a| 
ノードで実行できるテストのリスト。
 a| 
文字列の配列

|===


== 要求例

このメソッドの要求例を次に示します。

[listing]
----
{
   "method": "ListTests",
   "params": {},
   "id" : 1
}
----


== 応答例

このメソッドの応答例を次に示します。

[listing]
----
{
  "id": 1,
  "result": {
   "tests": [
             "TestConnectEnsemble",
             "TestConnectMvip",
             "TestConnectSvip",
             "TestDrives",
             "TestHardwareConfig",
             "TestLocateCluster",
             "TestPing",
             "TestLocalConnectivity",
             "TestRemoteConnectivity",
             "TestNetworkConfig"
           ]
      }
}
----
