= loggingServer の場合
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
loggingServer オブジェクトには、ストレージ クラスター用に構成されたログ ホストに関する情報が格納されます。使用 `GetRemoteLoggingHosts`現在のログホストを表示し、 `SetRemoteLoggingHosts`現在のログホストと新しいログホストの必要なリストを設定します。



== オブジェクトメンバー

このオブジェクトのメンバーは次のとおりです。

|===
| 名前 | 説明 | を入力します 


 a| 
ホスト
 a| 
ログサーバーのFQDNまたはIPアドレス
 a| 
文字列



 a| 
ポート
 a| 
ログサーバーとの通信に使用するポート番号
 a| 
整数

|===
