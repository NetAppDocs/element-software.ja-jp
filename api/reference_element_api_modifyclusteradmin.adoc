= ModifyClusterAdmin
:icons: font
:imagesdir: ../media/


[role="lead"]
「 ModyClusterAdmin 」メソッドを使用して、クラスタ管理者、 LDAP クラスタ管理者、またはサードパーティのアイデンティティプロバイダ（ IdP ）クラスタ管理者の設定を変更できます。管理者のクラスタ管理者アカウントのアクセス権を変更することはできません。

== 戻り値

このメソッドには戻り値はありません。



== 要求例

このメソッドの要求例を次に示します。

[listing]
----
{
   "method": "ModifyClusterAdmin",
   "params": {
     "clusterAdminID"  : 2,
     "password"   : "7925Brc429a"
   },
   "id" : 1
}
----
