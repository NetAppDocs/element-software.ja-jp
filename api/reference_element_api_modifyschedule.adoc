= ModifySchedule の追加
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
「変更スケジュール」を使用して、スケジュールされたスナップショットが発生する間隔を変更できます。スケジュールを削除または一時停止することもできます。

----

  "method": "ModifySchedule",
  "params": {
    "scheduleName" : "Chicago",
    "scheduleID" : 3
    },
  "id": 1
}
----
