= TestLocalConnectivity のように表示され
:icons: font
:imagesdir: ../media/


[role="lead"]
TestLocalConnectivity メソッドを使用すると、アクティブなクラスタ内の各ノードのクラスタ IP （ CIP ）に対して ping を実行できます。



== パラメータ

このメソッドには入力パラメータはありません。



== 戻り値

このメソッドの戻り値は次のとおりです。

|===
| 名前 | 説明 | を入力します 


 a| 
詳細
 a| 
ローカルのアクティブなクラスタ内の各ノードに対する、個々の ping 応答時間。
 a| 
JSON オブジェクト

|===


== 要求例

このメソッドの要求例を次に示します。

[listing]
----
{
    "method": "TestLocalConnectivity",
    "params": {},
    "id": 1
}
----
