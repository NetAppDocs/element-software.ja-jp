= Element ストレージシステムをセットアップする
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
NetApp Element ソフトウェアストレージシステムのセットアップでは、ストレージノードを設定し、ストレージクラスタを作成し、クラスタにドライブを追加します。Fibre Channel ネットワークを使用する場合は、 Fibre Channel ノードを設定できます。
