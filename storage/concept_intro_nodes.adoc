= ノード
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ノードは個別のハードウェアコンポーネントで、クラスタにグループ化され、ブロックストレージとしてアクセスされます。SolidFire ストレージシステムには、ストレージノードと Fibre Channel ノードの 2 種類のノードがあります。
