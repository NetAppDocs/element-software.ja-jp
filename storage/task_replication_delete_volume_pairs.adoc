= ボリュームペアを削除します
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
2 つのボリューム間のペア関係を解除するには、ボリュームペアを削除します。

. [ * データ保護 * > * ボリュームペア * ] を選択します。
. 削除するボリュームペアの操作アイコンをクリックします。
. [ 削除（ Delete ） ] をクリックします。
. メッセージを確認します。
