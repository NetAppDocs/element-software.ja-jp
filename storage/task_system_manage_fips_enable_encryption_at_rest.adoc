= 保存データの暗号化を有効にします
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
保存データの暗号化をクラスタ全体で有効または無効にすることができます。この機能はデフォルトでは有効になっていません。FIPS ドライブをサポートするには、保存データの暗号化を有効にする必要があります。

. NetApp Element ソフトウェア UI で、 * クラスタ * > * 設定 * をクリックします。
. [ 保存データの暗号化を有効にする ] をクリックします。 *
